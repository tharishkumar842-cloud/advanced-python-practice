python Comprehension Practice

This repository contains Python list and dictionary comprehension examples designed for interview preparation and AI/ML-focused Python practice.

Code 1: Replace Multiples of 3

Creates a new list where numbers divisible by 3 are replaced with the string "THREE", while other numbers remain unchanged.

Concepts:
List comprehension, conditional expression, modulus operator

Code 2: Remove Vowels from a Sentence

Iterates through a sentence and removes all vowels, storing only non-vowel characters in a list.

Concepts:
List comprehension, string iteration, conditional filtering

Code 3: Alternate Character Case

Uses index-based logic to convert characters at odd indexes to uppercase and even indexes to lowercase.

Concepts:
enumerate(), list comprehension, index-based condition

Code 4: Dictionary Comprehension with Condition

Creates a dictionary where keys are words and values are their lengths, including only words with length greater than 2.

Concepts:
Dictionary comprehension, conditional filtering, len() function

Code 5: Double Values at Even Indexes

Processes a list and doubles elements that are located at even indexes.

Concepts:
enumerate(), list comprehension, index-based filtering
