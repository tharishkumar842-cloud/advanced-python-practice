### Code 1: Replace Multiples of 3
Creates a new list where numbers divisible by 3 are replaced with the string "THREE", while other numbers remain unchanged.

**Concepts:**
- List comprehension
- Conditional expression
- Modulus operator

### Code 2: Remove Vowels from a Sentence
Iterates through a sentence and removes all vowels, storing only non-vowel characters in a list.

**Concepts:**
- List comprehension
- String iteration
- Conditional filtering

### Code 3: Alternate Character Case
Uses index-based logic to convert characters at odd indexes to uppercase and even indexes to lowercase.

**Concepts:**
- enumerate()
- List comprehension
- Index-based condition

### Code 4: Dictionary Comprehension with Condition
Creates a dictionary where keys are words and values are their lengths, including only words with length greater than 2.

**Concepts:**
